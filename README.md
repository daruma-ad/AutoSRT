# 🎬 AutoSRT — 動画編集前処理自動化Webアプリケーション

Premiere Pro 等での動画編集作業で最も辛い作業を効率化するための Web アプリケーションです。
ブラウザ上で **無音カット（FCP XML 生成）** と **LLM（Gemini 2.5 Flash）による「崩れない」SRT字幕校正** を行えます。

---

## 🚀 最新機能ハイライト

### フェーズ1: インテリジェント無音カット（FCP XML）
- 音声ファイル（MP3/WAV）からRMSベースで無音区間を高精度検出。
- **NTSC（29.97fps）累積ドリフトの完全解消**: 長尺動画でも後半での「音ズレ」を1フレームも許さない、厳密な `pproTicks` 計算を実装。
- **3つのカットプリセット**: ジェットカット・スタンダード・シネマティックをワンクリックで切替。
- パラメータ（音量閾値・最小無音時間・パディング）の直感的なスライダー調整。

### フェーズ2: コンテキスト考慮型 & スマート・フォーマット SRT 校正
- 最新LLM **Gemini 2.5 Flash** による、動画のジャンルや専門用語を理解した超高品質な字幕校正。
- **スマートフォーマッティング機能**: 細切れすぎる字幕（「あのー」等）の文脈を壊さない自動結合と、テレビ品質の「最大2行」自動改行制御。
- **究極の多重ガードレール**: LLM特有の「改行フォーマット違反」「勝手な字幕削除」「インデックス1からの強制リセット」に対抗する、自動修復（正規表現オートリペア）とオフセット逆算マッピングを搭載。タイムビデオとの同期を**絶対に破壊させません**。

## 📦 セットアップとインストール

### 前提条件
- Python 3.9 以上
- ffmpeg（音声処理の根幹として必須システム要件）

### インストール手順

```bash
# リポジトリをクローン または フォルダへ移動
cd 20260407autosrt

# 仮想環境を作成（推奨）
python -m venv .venv
source .venv/bin/activate  # macOS/Linux の場合
# .venv\Scripts\activate   # Windows の場合

# 依存パッケージのインストール
pip install -r requirements.txt
```

### ffmpeg のインストール（未インストールの場合）
```bash
# macOS
brew install ffmpeg

# Windows (chocolatey)
choco install ffmpeg
```

## 💡 使用上のプロフェッショナルな注意点

### 無音カット機能
- Premiere Pro で「ファイル > 書き出し > Final Cut Pro XML」から出力したオリジナルのXMLを使用してください。
- トーク系は「最小無音時間 300〜800ms」、ゲーム実況は「500〜1500ms」が目安です。

### SRT 字幕校正機能
- **同期ズレ防止の掟**: Premiere などの動画編集ソフトで「無音カット（XML）」を適用して**長さを短くした後のタイムライン**に対して、「未カットのフル音源から抽出した SRT」をインポートすると、当然ながら時間は合いません。必ず **「動画を切る前に抽出したSRTを校正し、それを組み込んだ上でXMLでカットする」** か、あるいは **「カット適用後の書き出し動画から改めてSRTを作成してから当システムに通す」** ようにしてください。
- 出力されたSRTファイルは、画面上の「🖥️ デスクトップに保存」ボタンを押すことで、Macのデスクトップ等に直接一瞬で保存され、すぐPremiereへD&D可能です。

## 🔧 技術詳細と「多重ガードレール」について

LLMに字幕データを扱う際のもっとも致命的なバグが「タイムスタンプの崩壊」です。本システムは以下のフェーズで堅牢性を確保しています。

1. **正規表現オートリペア**: LLMが改行を忘れ、日本語テキストと次のインデックス番号を密着させた場合（例：`テキスト281\n00:09...`) 、独自に `[^\n\d]` をフックして改行を即時補填。
2. **タイム印のオフセット逆算**: AIが一部の字幕を消滅させ、番号をリセットしても、`HH:MM:SS` スタンプを解析し本来のチャンク内の位置を特定・上書きします。
3. **最終フォールバック**: 照合不能レベルまでAIがテキストを狂わせた場合は、その区間のみ安全重視で「元の未校正テキスト」へ自動退避させ、**映像と音の同期決壊を100%防ぎます**。
