# 🚀 AutoSRT 導入ガイド（システム使用者向け）

このフォルダを受け取った方が、ご自身のパソコン（Mac または Windows）で AutoSRT を起動し、毎秒単位のジェットカットと精緻なAI字幕校正を使い始めるための手順書です。

このツールには、最新のAIモデル「Gemini 2.5 Flash」による**文脈考慮型のスマートフォーマット（TV級の自動改行・結合）**機能と、NTSCフレーム単位のズレを完全に防ぐ**1/254016000000秒単位のXML処理エンジン**が搭載されています。

---

## 1. 準備するもの（初回セットアップ）

アプリを動かすための「土台」として、以下の3つが必要です。

### ① Python のインストール（必須）
Python 3.9 以降が必要です。[Python公式サイト](https://www.python.org/downloads/) からご自身のOSに合わせたインストーラーをダウンロードし、実行してください。
⚠️ **重要（Windowsの方）**: インストール画面の最初にある「**Add Python to PATH**」に必ずチェックを入れてください！これを忘れると動きません。

### ② FFmpeg のインストール（必須）
音声処理のエンジンとなるツールです。
- **Macの場合:**
  [Homebrew](https://brew.sh/ja/) をインストール後、ターミナルで `brew install ffmpeg` を実行します。
- **Windowsの場合:**
  [FFmpeg Builds](https://github.com/BtbN/FFmpeg-Builds/releases) から `ffmpeg-master-latest-win64-gpl.zip` などをダウンロードし、中の `bin` フォルダをCドライブ等に配置後、「環境変数のPATH」に追加してください。（詳細は「Windows FFmpeg インストール」等で検索してください）

### ③ Gemini API キー（必須・無料）
字幕校正AIの脳として使います。
[Google AI Studio](https://aistudio.google.com/apikey) にGoogleアカウントでログインし、「Create API key」ボタンから文字列を取得してコピーしておきます。

---

## 2. アプリの初期構築（初回のみ）

このフォルダをデスクトップ等に解凍します。
ターミナル（Windowsの場合は「コマンドプロンプト」や「PowerShell」）を開き、このフォルダの中に移動（`cd`コマンド）します。

その後、以下の魔法のコマンドを入力し、エンターを押します。

```bash
# アプリケーションを動かすための外部パーツ（ライブラリ）をインストールします
pip install -r requirements.txt
```
※完了するまで数十秒〜数分お待ちください。

---

## 3. アプリの起動（普段の使い方）

準備ができたら、ターミナル（同じくこのフォルダの階層）で以下のコマンドを実行します。

```bash
streamlit run app.py
```

数秒後、**自動的にブラウザが立ち上がり**、美しいデザインの操作パネルが表示されます！

1. 画面左側の設定パネル「🔑 Gemini API 設定」に、取得した API キーを入力します。
2. あとは画面の案内に従って、PremiereのXMLファイルや録音データ（MP3等）、SRTファイルをアップロードするだけです。
3. 処理が終わったら「🖥️ デスクトップに保存」ボタンを押すことで、ダウンロードフォルダを探す手間なく即座にPremiereへのインポート準備が整います。

## 💡 プロの編集現場におけるアドバイス

- **映像とAI字幕のズレを防ぐ鉄則**: 
  「無音カット（フェーズ1）」を行って動画自体を短くする前に、**「フル尺の動画から抽出したSRTを、まずは校正（フェーズ2）にかける」** ようにしてください。カット済みの短い映像に対して、カット前の長いSRTを貼り付けると、タイムスタンプが完全にズレてしまいます。まずは字幕を整え、そこにカットを挟むか、または「完全にカットを終えた後の素材」から抽出したSRTを使うようにしてください。
